SQL is a computer language that we can use to interact with the database.


SQL is an interactive query language. Users type SQL commands into an interactive 
SQL program to retrieve data and display it on the screen, providing a convenient, 
easy-to-use tool for ad hoc database queries. 
 
 
SQL is a database programming language. Programmers embed SQL commands into 
their application programs to access the data in a database. Both user-written 
programs and database utility programs (such as report writers and data entry tools) 
use this technique for database access.

PK and FK
---------

primary key
-----------

primary key is used to identify the each record of the table
the main purpose of the primary is when we try to fetch the record from the table we will pass the primary key main argument to fetch the record among
all the available records

each row(record) in a table will contains the primary key

if you mention any field(attribute or property) as a primary key then we can't duplicate that key throughout the table records

we can't generate the primary key manually or we can make Sql to generate the PK automatically 

when say automatically we will use auto-increment in Sql


Foreign Key
-----------

foreign key is used to identify the other tables record inside the current table 
example
in student table we have a field called fee_contribution_id which we marked as foreign key then the same field 
the the student_fee_contribution table will act like a primary key

here in the student table we use fee_contribution_id to identify the other table(student_fee_contribution) table records 

the main purpose of the foreign key is to simplify the tables structures 
in simple terms we call this a normalization process( deviding the big table into multiple tables and creating the relationship between them using FK)
